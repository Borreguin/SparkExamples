Use the following steps to submit this application. 
Execute all steps in the spark-application directory through the terminal.

Step 1: Download Spark Ja

Spark core jar is required for compilation, therefore, 
download spark-core_2.10-1.3.0.jar from the following link Spark core jar 
and move the jar file from download directory to spark-application directory.
http://mvnrepository.com/artifact/org.apache.spark/spark-core_2.10/1.3.0


Step 2: Compile program

Compile the above program using the command given below. This command 
should be executed from the spark-application directory. 
Here, /usr/local/spark/lib/spark-assembly-1.4.0-hadoop2.6.0.jar 
is a Hadoop support jar taken from Spark library.
